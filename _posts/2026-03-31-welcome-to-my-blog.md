# 欢迎来到我的 GitHub 博客！

很高兴您能访问我的博客。这是我使用 GitHub Pages 和 Jekyll 创建的第一个博客。

## 为什么选择 GitHub Pages？

GitHub Pages 是一个完全免费的静态网站托管服务，具有以下优点：

1. **完全免费** - 无需支付任何费用
2. **无需服务器** - GitHub 负责所有托管
3. **自动部署** - 每次推送代码都会自动构建和部署
4. **支持自定义域名** - 可以使用自己的域名（可选）
5. **HTTPS 支持** - 自动提供 SSL 证书

## 博客功能

这个博客包含以下功能：

- 📝 **Markdown 支持** - 使用简单的 Markdown 语法编写文章
- 🏷️ **分类和标签** - 方便组织文章
- 🔍 **搜索功能** - 通过 GitHub 的搜索功能查找文章
- 📱 **响应式设计** - 在手机和电脑上都能良好显示
- 🎨 **简洁主题** - 使用 Minimal 主题，专注内容

## 如何使用这个博客？

1. 在 `_posts` 目录下创建新的 Markdown 文件
2. 文件名格式：`YYYY-MM-DD-title.md`
3. 在文件开头添加 YAML front matter
4. 使用 Markdown 语法编写内容
5. 推送更改到 GitHub

## 示例文章结构

```markdown
---
layout: post
title: "文章标题"
date: YYYY-MM-DD HH:MM:SS +0800
categories: 分类1 分类2
tags: 标签1 标签2
---

# 标题

文章内容...
```
